Building Blocks

Every node in a workflow is a tool that does one thing. Here are all the tools available, grouped by the kind of work they do.

Where Data Comes In

These nodes start your workflow — they’re how you get information into the pipeline.

📝
Text Input
Type or paste text. This is the most common starting point — paste a clinical note, a question, or any text you want to process.
📄
Document Upload
Upload a PDF, Word doc, or text file. The workflow reads and extracts the text automatically.
🎤
Voice Input
Use a dictation transcript as input. Pairs well with Lumen's dictation feature.
🌐
Web Scrape
Fetch and parse content from a URL. Useful for pulling in guidelines or reference material from the web.
Pro

Text Analysis

These nodes read text and extract useful structure from it.

📋
Summarize
Condense long text into a shorter summary. You control the length.
📄
Extract Entities
Find medications, diagnoses, procedures, and lab values in unstructured text.
🏷
Classify
Sort text into categories you define. Example: classify a note as 'urgent', 'routine', or 'follow-up'.
📈
Sentiment
Detect whether text is positive, negative, or neutral — with a confidence score.
🌐
Translate
Translate text between languages. Useful for multilingual patient communication.
🔑
Keywords
Extract the most important terms from a piece of text.
❓
Q&A
Answer a specific question based on the text that comes in. Example: 'What medications is this patient on?'
✏
Generate Text
Write new text based on a prompt. Example: 'Write a patient-friendly explanation of these lab results.'

Knowledge & Research

These nodes search databases and bring in external evidence.

📚
Search Clinical Knowledge
Search Lumen's clinical database for relevant guidelines and evidence.
📁
Search Your Documents
Search your personally uploaded files for relevant passages.
📚
PubMed Search
Search PubMed for peer-reviewed articles. Set the query and max results.
Pro
💊
FDA Drug Labels
Look up FDA-approved prescribing information — indications, warnings, dosing.
Pro
🧪
Clinical Trials
Search ClinicalTrials.gov for active and completed studies.
Pro

Medications

💊
Identify a Drug (RxNorm)
Normalize drug names — brand to generic, get drug class and RxCUI.
Pro
⚠
Check Drug Interactions
Check for interactions between drugs. Returns severity and management.
Pro
💧
IV Drip Calculator
Calculate infusion rates from weight-based dosing.
Pro
🩸
Renal Dose Check
Compute CrCl and look up renal dose adjustments.
Pro

Coding & Labs

🏥
ICD-10 Lookup
Find diagnosis codes from a clinical description.
Pro
🧾
CPT Lookup
Find procedure codes.
Pro
📋
HCPCS Lookup
Find Level II codes for DME, supplies, and drugs.
Pro
🧪
Lab Reference Ranges
Get normal ranges for any lab test.
Pro
🧮
Clinical Calculator
Run validated scoring tools — eGFR, CHA₂DS₂-VASc, MELD, Wells, and more.
Pro
✅
CMS Coverage
Check Medicare coverage determinations (NCDs and LCDs).
Pro

Protocols & Decisions

☑
Clinical Checklist
Load evidence-based protocol checklists (sepsis, DKA, central line, etc.).
Pro
🏥
Triage Assessment
Score patient acuity using ESI, Manchester, CTAS, or START.
Pro
🧠
Differential Diagnosis
Generate a ranked differential with workup recommendations.
Pro
🦠
Antimicrobial Selector
Select appropriate antibiotics with dosing and allergy alternatives.
Pro
📖
Clinical Guidelines
Search USPSTF and current practice guidelines.
Pro

Where Results Go Out

These nodes deliver the final output of your workflow.

💬
Send to Chat
Output appears in the chat — the most common output node.
📄
Export as PDF
Generate a downloadable PDF document.
Pro
📊
Export as Slides
Generate a PowerPoint presentation.
Pro
📋
Create Spreadsheet
Generate an Excel workbook.
Pro
📧
Send Email
Email the output to a specified address.
Max
💡
Pro and Max badges indicate which subscription tier is needed for that tool. Free-tier users can still build workflows with basic text analysis and output nodes.
