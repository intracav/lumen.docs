Getting Started

If you can send a text message, you can use Lumen. Pick a scenario below and follow along — each one takes less than a minute to try.

💬

Your First Chat

Open the app, type a question, get an answer. A 60-second walkthrough of the basics.

"What's the first-line treatment for new-onset A-fib?"
Start here →
⚠

Check a Drug Interaction

You're about to add a new med. Ask Lumen if it's safe with what the patient is already on.

"Can I give ketorolac if the patient is on apixaban?"
Try it →
🏥

Find a Billing Code

Describe the diagnosis or procedure in your own words. Lumen finds the right ICD-10 or CPT code.

"ICD-10 for acute on chronic diastolic heart failure"
Try it →
🧠

Build a Differential

List symptoms and patient details. Lumen generates a ranked differential with workup recommendations.

"Differential for a 28yo with acute pleuritic chest pain and tachycardia"
Try it →
📝

Draft a Clinical Note

Tell Lumen what happened in the visit. It creates a SOAP note, discharge summary, or referral letter you can review and sign.

"Write a SOAP note for a 55yo with uncontrolled HTN, BP 168/96, added amlodipine 5mg"
Try it →
🧮

Run a Clinical Calculator

Need a CHA₂DS₂-VASc, MELD, eGFR, or Wells score? Just describe the patient and Lumen calculates it.

"CHA2DS2-VASc for a 72yo woman with HTN and diabetes"
Try it →
