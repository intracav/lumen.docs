Codes, Labs & Calculators

Nobody went to medical school to memorize billing codes. Describe the diagnosis or procedure in your own words and Lumen finds the right code. It also checks lab reference ranges, verifies Medicare coverage, and runs clinical calculators like CHA₂DS₂-VASc and MELD.

Available Tools

🏥
ICD-10 Lookup
Search ICD-10-CM diagnostic codes. Matches clinical descriptions to specific codes with laterality, severity, and sequencing guidance.
"ICD-10 code for acute anterolateral STEMI, initial encounter"
Coding
🧾
CPT Lookup
Find CPT procedure codes. Resolves procedure descriptions to billing codes with intensity levels and component breakdowns.
"CPT code for laparoscopic cholecystectomy"
Coding
📋
HCPCS Lookup
Search HCPCS Level II codes for DME, supplies, and non-physician services. Covers J-codes for drugs, E-codes for equipment, and more.
"HCPCS code for a standard wheelchair"
Coding
✅
CMS Coverage Lookup
Check Medicare coverage determinations — both National (NCDs) and Local (LCDs). Returns coverage criteria and required documentation.
"Does Medicare cover continuous glucose monitoring for type 2 diabetes?"
Insurance
🧪
Lab Reference Ranges
LOINC-mapped laboratory reference ranges. Returns normal intervals adjusted for units, age, and sex, with clinical significance notes for abnormal values.
"What is the normal reference range for TSH?"
Labs
🧮
Medical Calculator
Validated clinical scoring tools — eGFR, CHA₂DS₂-VASc, MELD, CURB-65, Wells criteria, APACHE II, and more. Returns scores with interpretation.
"Calculate MELD-Na score: bilirubin 3.2, INR 1.8, creatinine 1.5, sodium 131"

Common Medical Calculators

Lumen supports dozens of validated scoring tools. Here are the most frequently used:

Calculator Used For
CHA₂DS₂-VASc Stroke risk in atrial fibrillation
HAS-BLED Bleeding risk on anticoagulation
MELD / MELD-Na Liver disease severity
eGFR (CKD-EPI) Kidney function estimation
CURB-65 Pneumonia severity
Wells Criteria DVT/PE probability
APACHE II ICU mortality prediction
Glasgow Coma Scale Neurological assessment
SOFA Score Organ dysfunction in sepsis
BMI Body mass index
💡 Tip
Just ask naturally
You don’t need to know the calculator name. Just describe what you need: “What’s the stroke risk for my AFib patient?” and Lumen will select CHA₂DS₂-VASc automatically.
