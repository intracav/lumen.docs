Patient Records & Notes

If your practice has EHR integration set up, Lumen can pull a patient’s current medications, recent labs, and diagnoses right into the conversation. It can also draft clinical notes and check scheduling availability.

⚠️
EHR integration requires setup. These tools are only available after configuring your FHIR connection in Settings > EHR Integration. See your administrator for access.

Available Tools

👤
FHIR Patient Data
Queries your EHR via FHIR R4. Pulls current medications, recent labs, active diagnoses, allergies, and encounter history for a specific patient.
"Pull the current medication list for patient ID 12345"
EHR
📅
Scheduling
Queries the booking system for available appointment slots. Filter by provider, specialty, date range, and location.
"Show available cardiology slots next week"
EHR
📝
Clinical Notes
Auto-generates clinical documentation — SOAP notes, H&P notes, discharge summaries, and referral letters. Structured according to standard clinical formats.
"Generate a SOAP note for today's visit — 45yo with uncontrolled HTN"
Documentation

Clinical Note Generation

Lumen can draft several types of clinical documentation:

Note Type Best For
SOAP Note Routine outpatient visits
H&P Admission history and physical
Discharge Summary Hospital discharge documentation
Referral Letter Specialist referral correspondence
Progress Note Inpatient daily progress
