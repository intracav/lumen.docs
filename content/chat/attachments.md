Have a research paper you want summarized? A lab report you want interpreted? A protocol you want compared to current guidelines? Just attach the file to your chat and ask away. Lumen reads the document and uses it to answer your question.

## How to Attach Files

<div class="steps-container">

{{< step num="1" title="Click the paperclip icon" >}}
In the input bar, click the **paperclip icon** on the left side. This opens the document tray.
{{< /step >}}

{{< step num="2" title="Choose or upload a file" >}}
You can either:
- **Select from existing documents** already in your Lumen library
- **Upload a new file** by dragging it into the tray or clicking the upload area
{{< /step >}}

{{< step num="3" title="Watch the upload progress" >}}
When you upload a new file, you'll see four progress stages:
1. **Uploading** — Transferring the file
2. **Extracting** — Reading the text content
3. **Embedding** — Creating searchable indexes
4. **Done** — Ready to use
{{< /step >}}

{{< step num="4" title="Ask your question" >}}
Once attached, the file appears as a chip above the text field. Now type your question about the document and send.
{{< /step >}}

</div>

## Supported File Types

| Format | Max Size | Description |
|--------|----------|-------------|
| **PDF** | 5 MB | Research papers, guidelines, reports |
| **DOCX** | 5 MB | Word documents, clinical notes |
| **TXT** | 5 MB | Plain text files |
| **CSV** | 5 MB | Data tables, lab results |

{{< tip title="Multiple attachments" >}}
You can attach multiple documents to a single message. Lumen will consider all of them when generating its response. Document chips appear in a row above the text field.
{{< /tip >}}

{{< callout type="warning" >}}
**Privacy reminder:** Documents you upload are stored securely in your personal library. They are only accessible to you and are not shared with other users.
{{< /callout >}}
